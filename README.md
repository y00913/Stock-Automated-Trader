# Stock-Automated-Trader
# 주식 자동 매매 프로그램



수원대학교 컴퓨터SW 안유태 & ![김규민](https://github.com/kgm-930)

캡스톤 디자인 프로젝트 결과물. visual C# console & 키움증권 openAPI 활용. 

키움증권 openAPI, 계정, 영웅문 필요.

# 기능

- 선택 종목 조회
- 평가 금액 조회
- 손익 계산 후 자동 거래
- 실 거래 전 테스트
